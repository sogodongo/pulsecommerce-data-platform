Bronze Layer

Raw, immutable, append-only. Written by Flink within 90s of Kafka offset commit. Partitioned by event_date/hour.

Silver Layer

Deduplicated, PII-masked, enriched. MERGE upserts every 15 minutes via Glue Spark. SCD Type 2 for orders.

Gold Layer

Kimball star schema. dbt incremental models. Refreshed hourly. Powers Redshift, Athena, and FastAPI.

Flink Applications

bronze_writer, session_stitcher, fraud_scorer, churn_enrichment. RocksDB state. Exactly-once semantics.

ML Pipeline

23-feature behavioral Feature Store. XGBoost churn model. Weekly retrain with AUC-ROC gate of 0.80.

Orchestration

MWAA Airflow 2.8. Three DAGs: silver_refresh (30-min), gold_models (triggered), churn_retrain (weekly).
